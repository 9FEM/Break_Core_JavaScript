# Break_Core_JavaScript

Core JavaScript 도서 뿌수기

## 스터디 알아두기

- 스터디는 일주일 1회 진행
- 스터디 시간은 대략 1시간 ~ 1시간 30분 진행
- 코어자바스크립트 책을 중점으로 모던 자바스크립트 책을 참고하기

## 스터디 방식

- 코어 자바스크립트 책 소목차로 나누어 진행
- 2-3명이 조를 이루어 진행
- 이번주 읽은 챕터는 퀴즈 2문제씩 준비
- 지난번 읽은 챕터는 조끼리 발표 준비
